Reasonable Research

How it works

This app goes through your CSV database and allows you to interactively set boolean flags on fields in a table depending on other values in the same row.

Currently only simple case-insensitive and diacritic-blind search is supported.

Config file format

The config is a CSV file. It has the following format:

SOURCE_COLUMN;RESULT_COLUMN;SEARCH_PHRASE

You can include more keyphrases like so:

SOURCE_COLUMN;RESULT_COLUMN;SEARCH_PHRASE;SEARCH_PHRASE_2;SEARCH_PHRASE_3

Database format

Database is expected to be a CSV file with header. For every row in the config file, the script will look in the 
$SOURCE_COLUMN
for text matching the 
$SEARCH_PHRASE
and depending on the answer (yes or no), it'll set 1 or 0 in the 
$RESULT_COLUMN
